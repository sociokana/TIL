### 변수 선언


```
package ch01;

public class Vars {
	public static void main(String[] args) {
		int hoge1, hoge2; 
		hoge1 = 10;
		hoge2 = 15; 
		hoge3 = 20; 
		
		int hogeBin = 0b1010; 
		int hogeOct = 01010; 
		int hogeHex = 0x1010; 

		long hogeLong = 12345678900L;
		
		System.out.println("hoge1: " + hoge1);
		System.out.println("hoge2: " + hoge2);
		System.out.println("hoge3: " + hoge3);
		System.out.println("Bin: " + hogeBin);
		System.out.println("Oct: " + hogeOct);
		System.out.println("Hex: " + hogeHex);
		System.out.println("Long: " + hogeLong);		
	}	
}
```

- 변수의 이름
	- 영문자 및 \_과 \$ 사용 가능
	- 소문자로 시작하고 Camel Rotation 을 사용.
	- 숫자로 시작할 수는 없음 
	- ``reserved words`` 는 사용할 수 없음. 
	- 나중에 봐도 알기 쉽도록 작성. 

#### 자료형
- 문자형
	- ``char`` (2)
	
- 정수형
	- ``byte`` (1), ``short`` (2), ``int`` (4), ``long`` (8)
	- 가장 왼쪽의 비트는 부호 비트
	- ``byte``: 1바이트 단위의 자료형 동영상, 음악 파일, 실행 파일의 자료를 처리할 때 사용
	- ``short``: 2바이트 단위의 자료형 C/C++ 언어와 호환 시 사용
	- ``int``: 정수에 대한 기본 자료형. 
	- ``long``: 숫자의 뒤에 알파벳 L이나 l을 써서 long형임을 표시. 숫자 (리터럴) 는 이렇게 따로 지정을 하지 않을 경우 자동으로 4바이트로 잡힘. 
	
- 실수형
	- ``float`` (4), ``double`` (8)
	- 실수는 기본적으로 ``double`` 이 됨. 실수 뒤에 f를 쓰면 ``float`` 으로 처리할 수 있음. 
	- 부동 소수점 방식으로 처리. 
		- 가수 x (밑수 ^ 지수) 로 나누어서 처리. 밑수는 2. 
		- 정규화 할 경우 가수부분 첫 번째 자리 숫자는 항상 1이 됨. 따라서 가수는 항상 1.x 
		- 정규화: 가수가 밑수보다 작은 한 자리까지 가수로 표현되는 것. 
		- 오차: 지수부가 0을 표현할 수 없기 때문에 오차가 발생함. 

- 논리형
	- ``boolean`` (1)
